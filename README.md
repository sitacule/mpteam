### 1단계: 파일 열기
```
C:\Users\LOTTE\Desktop\업무보고 자동화\index.html
```
Chrome 또는 Edge 브라우저로 열기

### 2단계: 초기 설정 (팀장만 1회)

**초기 설정 화면이 나타나면:**

#### ✅ 저장 방식 선택 (3가지 중 선택)
```
💾 로컬 저장        ← PC에만 저장 (팀 공유 불가)
📁 공유폴더        ← ⭐ 권장! 회사 공유폴더 사용
☁️  Firebase      ← 클라우드 동기화 (별도 설정 필요)
```

**🔴 공유폴더 모드 선택이 가장 간편합니다:**
- 별도 서버 필요 없음
- 회사 공유폴더 (\\Server\Share 등) 직접 사용 가능
- 팀원이 파일 동기화로 자동 수신

#### 👥 팀 정보 입력
```
팀장 이름: 신해리
팀원 이름: 홍길동, 김철수, 이영희, 박재룡, 성연재, 김소영
팀장 비밀번호: (선택사항)
```

#### ✅ 설정 완료
- "설정 완료" 클릭
- **"🔗 공유 링크"** 모달 표시됨
- 공유 링크를 팀원들에게 카카오톡으로 전달하거나
- `team_config.json` 파일을 공유폴더에 저장

## 💼 팀원 사용법 (각 팀원)

### 1단계: 앱 진입
**방법 1) 팀장이 보낸 링크 클릭**
- 자동으로 모든 설정이 로드됨
- "팀원" 역할 선택 → 이름 클릭 → 입장

**방법 2) 공유폴더의 team_config.json 가져오기**
- 로그인 화면 → `📂 설정 가져오기` → `team_config.json` 선택

### 2단계: 공유폴더 연결
앱 로그인 후 헤더에 **"📁 폴더선택" 버튼** 클릭
```
회사 공유폴더 선택 (예: Z:\공유폴더\경영기획팀)
```
→ `● 공유폴더 연결: 경영기획팀` 표시됨

### 3단계: 업무 입력
**"📋 내 업무" 탭에서:**
- 업무명, 진행내용 입력
- 분야 선택 (회계/공시/청구/예산/자금/내부통제/IT...)
- 구분 선택 (중점/일상/익월)
- 진행률(%) 입력
- "✅ 등록" 클릭

### 4단계: 팀장에게 전송
**수정 완료 후:**
```
헤더 → [📤 전송] 클릭
```
→ 내 업무들이 공유폴더의 `홍길동_업무.json` 저장됨
→ 토스트 알림: "📤 전송 완료 (3건)"

### 5단계: 자동 동기화
```
헤더 → [🔄 동기화] 클릭
```
또는 **자동 30초마다 동기화**
→ 다른 팀원들의 최신 업무 수신
→ 토스트 알림: "🔄 업데이트: 김철수님(2건), 박재룡님(1건)"

## 👑 팀장 사용법

### 1단계: 초기 설정 (1회만)
위 "2단계: 초기 설정" 참고

### 2단계: 공유폴더 연결
앱 로그인 후 헤더 → **"📁 폴더선택"** 클릭
```
Z:\공유폴더\경영기획팀 선택
```

### 3단계: 팀원 데이터 동기화
```
헤더 → [🔄 동기화] 클릭
```
→ 모든 팀원의 `*_업무.json` 파일을 읽어서 통합
→ 팀전체 현황 자동 업데이트

### 4단계: 팀 전체 현황 확인
**"📋 전체 현황" 탭:**
- 팀원별 카드 (완료율, 상태)
- 기한초과 이슈 배너 (빨강)
- 전체 업무 테이블

**"📑 월간보고" 탭:**
- PPTX 형식으로 자동 생성
- [중점] [일상] [익월] 섹션
- 분야별([회계], [공시] 등) 자동 분류
- 인쇄/PDF 저장 가능

## 📤 공유폴더 동기화 상세

### 파일 구조
```
Z:\공유폴더\경영기획팀\
├── team_config.json          ← 팀 설정 (공유링크 대신)
├── 홍길동_업무.json         ← 홍길동의 업무 목록
├── 김철수_업무.json         ← 김철수의 업무 목록
├── 이영희_업무.json         ← 이영희의 업무 목록
└── (팀원수만큼...)
```

### 작동 방식
1. **팀원이 📤 전송 클릭**
   - `{팀원명}_업무.json` 파일 생성/업데이트
   - 동기화 시간(`syncedAt`) 기록

2. **팀장/다른팀원이 🔄 동기화 클릭**
   - 공유폴더의 모든 `*_업무.json` 읽기
   - 새로운 파일 감지 시 자동 추가
   - 업데이트된 파일 감지 시 알림 표시

3. **자동 동기화 (30초마다)**
   - 백그라운드에서 자동 실행
   - 새 데이터 감지 시만 알림
   - 옵션: 끌 수 있음

### 혼동 방지
- 각 팀원은 **자신의 파일만 수정**
- 팀장은 모든 파일 볼 수 있지만 수정하지 않는 것 권장
- 충돌 없음 (각자 다른 파일)

## 🐛 문제 해결

### Q: 공유 버튼이 보이지 않음
**A:** 초기 설정에서 **"📁 공유폴더" 선택** 필요
```
⚙️ 초기설정 → 저장 방식: 📁 공유폴더 → 설정 완료
```

### Q: 동기화가 안 됨
**A:** 공유폴더 연결 확인
```
헤더에 "● 공유폴더 연결" 표시 확인
없으면 → [📁 폴더선택] 클릭 → 폴더 다시 선택
```

### Q: 전송 완료 후 팀장이 못 봄
**A:** 팀장이 동기화 클릭 필요
```
[🔄 동기화] 클릭 → 전체 현황 탭 새로고침
```

### Q: 파일 권한 오류 (Permission denied)
**A:** 공유폴더 쓰기 권한 확인
```
공유폴더 우클릭 → 속성 → 보안 → 쓰기 권한 있는지 확인
없으면 IT팀에 문의
```

### Q: 팀 설정이 자꾸 초기화됨
**A:** 공유폴더에 team_config.json 저장
```
앱 헤더 → [🔗 설정공유] → [💾 폴더에 저장] 클릭
→ team_config.json이 공유폴더에 저장됨
→ 앱 삭제해도 팀 설정 유지됨
```
