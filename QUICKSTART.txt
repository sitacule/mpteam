## 2️⃣ 초기 설정 (팀장만 1회)
────────────────────────────────────────
화면: "⚙️ 초기 설정" 모달

Step 1) 저장 방식 선택
   └─ 📁 공유폴더 ← [클릭] ⭐ 회사 공유폴더 사용 권장

Step 2) 팀 정보 입력
   ├─ 팀장 이름: 신해리
   ├─ 팀원 이름: 홍길동, 김철수, 이영희, 박재룡, 성연재, 김소영
   └─ 팀장 비밀번호: (선택사항)

Step 3) "설정 완료" 클릭
   └─ "🔗 공유 링크" 모달 표시됨
   └─ 링크를 팀원 카톡으로 전송하거나
   └─ "💾 내보내기" 클릭 → team_config.json 공유폴더에 저장

## 3️⃣ 로그인 (팀원)
────────────────────────────────────────
방법 1) 팀장이 보낸 링크 클릭
   └─ 자동으로 모든 설정 로드
   └─ "팀원" 선택 → 자신 이름 클릭 → "입장"

방법 2) 공유폴더 이용
   └─ 로그인 화면 → "📂 설정 가져오기"
   └─ 공유폴더 → team_config.json 선택

## 4️⃣ 공유폴더 연결 (모두)
────────────────────────────────────────
로그인 후 헤더에서:

   [📁 폴더선택] 버튼 클릭
      ↓
   Z:\공유폴더\경영기획팀 (또는 귀사 공유폴더 경로)
      ↓
   헤더 변경: "● 공유폴더 연결: 경영기획팀" ✅

## 5️⃣ 업무 입력 (팀원)
────────────────────────────────────────
"📋 내 업무" 탭:

   [✅ 업무 추가] 버튼
      ↓
   ├─ 등록일: (자동)
   ├─ 업무명: "4월 결산 완료" *필수
   ├─ 진행내용: "월별 손익 분석 완료"
   ├─ 분야: [회계] ← 드롭다운
   ├─ 구분: [중점] ← 중점/일상/익월
   ├─ 진행률: 75% ← 0~100
   ├─ 완료예정일: 2026-04-30
   ├─ 우선순위: A ← A/B/C/D
   ├─ 처리여부: 진행중 ← 진행중/처리완료/보류
   └─ 비고: "IFRS 영향 검토 완료"
      ↓
   [✅ 등록]

## 6️⃣ 전송 (팀원이 매일/주기적으로)

## 7️⃣ 동기화 (팀원/팀장)
────────────────────────────────────────

## ✅ 체크리스트 (문제 해결)
────────────────────────────────────────
[ ] 브라우저: Chrome 또는 Edge 사용 중?
    └─ Firefox/Safari는 File System API 미지원

[ ] 공유폴더 보이는지? (헤더 좌측)
    ├─ "● 공유폴더 연결" → OK
    ├─ "● 미연결" → [📁 폴더선택] 클릭

[ ] 전송 버튼 보이는지?
    ├─ 안 보임 → 초기설정에서 📁 공유폴더 선택 했는지 확인
    └─ 보임 → OK, 업무 입력 후 [📤 전송] 클릭

[ ] 전송 후 팀장이 못 본다면?
    ├─ 팀장이 [🔄 동기화] 클릭 했는지 확인
    └─ 공유폴더 권한 확인 (읽기/쓰기)

## 🆘 자주 묻는 질문
────────────────────────────────────────

Q: 공유폴더를 잘못 선택했어요
A: 다시 [📁 폴더선택] 클릭 → 올바른 폴더 선택

Q: 다른 팀원 업무를 편집하고 싶어요
A: 팀원은 자신 업무만 편집 가능 (보기만 가능)
   팀장은 모든 업무 볼 수 있지만 자신 업무만 편집 권장

Q: 공유폴더 권한이 없다고 해요
A: IT팀에 문의 → 공유폴더 쓰기 권한 요청

Q: 파일을 삭제하면 어떻게 되나요?
A: 자신의 {이름}_업무.json만 삭제하면 됨
   다른 팀원 파일은 건드리지 말 것

Q: 설정을 다시 하고 싶어요
A: 로그아웃 → "⚙️ 설정 초기화" → 새 설정 시작
