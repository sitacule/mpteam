# 🔧 트러블슈팅 가이드

## 🚨 파일 열 때 File System Access API 오류

### 증상
```
showDirectoryPicker is not a function
또는
File System Access API가 지원되지 않습니다
```

### 원인
로컬 파일(`file://` 프로토콜)에서는 보안상 File System Access API 사용 불가

### 해결방법

#### 방법 1: 간단한 HTTP 서버 사용 (권장)
```bash
# Windows 기본 Python이 있으면 (Python 3.x)
python -m http.server 8000 --directory "C:\Users\LOTTE\Desktop\업무보고 자동화"

# 또는 PowerShell 사용
cd "C:\Users\LOTTE\Desktop\업무보고 자동화"
python -m http.server 8000

# 또는 npm이 설치되어 있으면
npx http-server . -p 8000
```

그 다음:
```
브라우저에서: http://localhost:8000
```

#### 방법 2: Chrome에서 로컬 파일 보안 비활성화 (임시방편)
```
Chrome 바로가기 우클릭 → 속성
대상 항목 끝에 추가:
  --disable-web-security --disable-gpu

예시:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --disable-web-security --disable-gpu --user-data-dir="C:\temp\chrome"

⚠️ 보안 위험 - 테스트 시에만 사용
```

## 🐛 주요 에러 & 해결방법

### 1️⃣ "공유폴더가 연결되지 않음" 표시
────────────────────────────────────
**증상:** 헤더에 "● 미연결" 표시, [📁 폴더선택] 버튼 활성화 안 됨

**확인사항:**
```
1. 초기설정 때 "📁 공유폴더" 모드 선택했는가?
   → 안 했으면: 로그아웃 → 설정 초기화 → 다시 설정

2. 브라우저가 Chrome/Edge인가?
   → Firefox, Safari는 File System API 미지원

3. HTTP 서버로 실행 중인가?
   → 로컬 파일(file://)은 API 사용 불가
   → 위 "간단한 HTTP 서버 사용" 참고
```

### 2️⃣ [📤 전송] 버튼이 안 보임
────────────────────────────────────
**증상:** 헤더에 전송 버튼이 없음

**확인:**
```
1. 초기설정 -> 📁 공유폴더 선택했는가?
   (로컬/Firebase 선택하면 전송 버튼 안 보임)

2. 로그인했는가?
   (로그인 전에는 헤더 자체가 안 보임)

3. 공유폴더가 연결되었는가?
   [📁 폴더선택] 클릭 -> 폴더 선택
```

### 3️⃣ 전송 클릭해도 공유폴더에 파일이 안 생김
────────────────────────────────────
**증상:**
```
[📤 전송] 클릭 → 토스트는 뜸 → 폴더 확인하면 파일 없음
```

**원인 & 해결:**
```
1. 폴더 권한 부족
   → 공유폴더 우클릭 → 속성 → 보안 → 쓰기 권한 확인
   → 없으면 IT팀 문의

2. 브라우저 권한 미부여
   → Chrome에서 "권한 부여" 팝업이 떴었는가?
   → 못 봤으면: [📁 폴더선택] 다시 클릭 → 권한 허용

3. 브라우저 캐시 문제
   → F12 → 개발자도구 → 콘솔에서:
     localStorage.clear();
     location.reload();
```

### 4️⃣ 동기화 버튼 클릭해도 업무가 안 보임
────────────────────────────────────
**증상:**
```
[🔄 동기화] 클릭 → "업데이트: ..." 토스트 뜸 → 탭에 안 보임
```

**확인:**
```
1. 탭을 변경해보자
   → [📋 내 업무] 탭으로 이동 → 새로고침 (F5)

2. 다른 팀원 파일이 공유폴더에 있는가?
   → 공유폴더 → {팀원명}_업무.json 존재 확인
   → 없으면 그 팀원이 [📤 전송] 아직 안 한 것

3. 권한 문제
   → 공유폴더 읽기 권한 확인
```

### 5️⃣ 한글이 깨짐 (인코딩 오류)
────────────────────────────────────
**증상:** 공유폴더 파일 내용이 "????" 같이 표시

**해결:**
```
대부분의 경우 자동으로 UTF-8 처리됨
문제 있으면:

1. 파일 우클릭 → 연결 프로그램 → 메모장
2. 메모장 상단 → 파일 → 다른 이름으로 저장
3. 인코딩: UTF-8 선택 후 저장
```

### 6️⃣ 설정 공유 링크가 작동 안 함
────────────────────────────────────
**증상:**
```
공유 링크 클릭 → 로딩 후 설정이 안 로드됨
```

**확인:**
```
1. URL이 긴지 확인
   (한글 때문에 매우 길 수 있음 - 정상)

2. 주소 복사 완벽히 했는가?
   헤더 → [🔗 설정공유] → [링크 복사] 클릭
   (수동 선택 말고 버튼 사용)

3. 브라우저 로컬스토리지 확인
   F12 → 개발자도구 → Application → Local Storage
   → wrs_cfg 항목이 있는가?
```

### 7️⃣ "이미 로그인되어 있습니다" 오류
────────────────────────────────────
**증상:** 로그인 화면이 안 나타남

**해결:**
```
1. 로그아웃 클릭 (헤더 우측)

2. localStorage 초기화
   F12 → 개발자도구 → 콘솔에서:
   localStorage.clear();
   location.reload();

3. 쿠키 삭제
   Chrome 설정 → 개인정보 보호 및 보안 
   → 쿠키 및 기타 사이트 데이터 삭제
```

### 8️⃣ 월간보고가 안 나타남
────────────────────────────────────
**증상:**
```
[📑 월간보고] 탭 클릭 → 빈 화면
```

**확인:**
```
1. 해당 월에 업무가 있는가?
   [📋 내 업무] 탭 → 업무 입력 확인
   (구분을 반드시 "중점"/"일상"/"익월" 중 선택)

2. 기준 월이 맞는가?
   월간보고 탭 → 기준 월 드롭다운 → 2026-05 같이 선택

3. 개발자 콘솔 에러 확인
   F12 → 콘솔 탭 → 빨간 에러 있는가?
```

## 🔍 개발자 콘솔로 디버깅

### 콘솔 열기
```
F12 또는 Ctrl+Shift+I (Chrome/Edge)
```

### 1) 설정 확인
```javascript
// 콘솔에 입력
cfg
// 출력: { storage: 'folder', leader: '신해리', members: [...], ... }
```

### 2) 현재 세션 확인
```javascript
session
// 출력: { role: 'member', name: '홍길동' }
```

### 3) 업무 데이터 확인
```javascript
tasks
// 출력: [ { id: '...', member: '홍길동', name: '4월 결산', ... }, ... ]
```

### 4) 공유폴더 핸들 확인
```javascript
sharedFolderHandle
// 출력: FileSystemDirectoryHandle { name: '경영기획팀', ... }
// null이면 폴더가 연결 안 됨
```

### 5) 토스트 메시지 직접 보기
```javascript
showToast('테스트 메시지', 'success')
// 우측 하단에 초록색 토스트 보임
```

### 6) localStorage 전체 내용 보기
```javascript
Object.keys(localStorage).forEach(key => console.log(key, localStorage[key]))
```

## 📊 진단 체크리스트

파일 열 때 아래를 확인하세요:

```
브라우저
☐ Chrome 또는 Edge 사용 중?
☐ 최신 버전인가? (chrome://settings/help 확인)

HTTP 서버 실행 여부
☐ http://localhost:8000 로 접근 중인가?
  (file:// 프로토콜 사용 중이면 안 됨)

초기 설정
☐ 📁 공유폴더 모드 선택했는가?
☐ 팀장명, 팀원명 입력했는가?

공유폴더 연결
☐ [📁 폴더선택] 클릭했는가?
☐ 유효한 폴더 선택했는가?
☐ 헤더에 "● 공유폴더 연결" 표시되는가?

로그인
☐ 팀원/팀장 역할 선택했는가?
☐ 자신 이름 선택했는가?

업무 입력
☐ [📋 내 업무] 탭 → [✅ 업무 추가] 클릭했는가?
☐ 필수 항목(업무명, 분야, 구분) 다 입력했는가?

전송
☐ [📤 전송] 버튼 보이는가? (빨간색)
☐ 클릭하면 토스트 메시지("📤 전송 완료 (N건)")가 뜨는가?
☐ 공유폴더에 {이름}_업무.json 파일이 생겼는가?

동기화
☐ [🔄 동기화] 버튼 보이는가?
☐ 클릭하면 토스트 메시지가 뜨는가?
☐ [📋 내 업무] 탭에 모든 업무가 보이는가?
```
